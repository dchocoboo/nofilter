    Properties:
    Name Type Attributes Default Description
    input string | Buffer <optional>
    		 null Input source data.
    inputEncoding BufferEncoding <optional>
    		 null Encoding name for input, ignored if input is not a String.
    highWaterMark number <optional>
    		 16384 The maximum number of bytes to store in the internal buffer before ceasing to read from the underlying resource. Default=16kb, or 16 for objectMode streams.
    encoding BufferEncoding <optional>
    		 null If specified, then buffers will be decoded to strings using the specified encoding.
    objectMode boolean <optional>
    		 false Whether this stream should behave as a stream of objects. Meaning that stream.read(n) returns a single value instead of a Buffer of size n.
    decodeStrings boolean <optional>
    		 true Whether or not to decode strings into Buffers before passing them to _write().
    watchPipe boolean <optional>
    		 true Whether to watch for 'pipe' events, setting this stream's objectMode based on the objectMode of the input stream.
    readError boolean <optional>
    		 false If true, when a read() underflows, throw an error.
    allowHalfOpen boolean <optional>
    		 true If set to false, then the stream will automatically end the writable side when the readable side ends.
    autoDestroy boolean <optional>
    		 true Whether this stream should automatically call .destroy() on itself after ending.
    defaultEncoding BufferEncoding <optional>
    		 'utf8' The default encoding that is used when no encoding is specified as an argument to stream.write().
    emitClose boolean <optional>
    		 true Whether or not the stream should emit 'close' after it has been destroyed.
    readableHighWaterMark number <optional>
    		 Sets highWaterMark for the readable side of the stream. Has no effect if highWaterMark is provided.
    readableObjectMode boolean <optional>
    		 false Sets objectMode for readable side of the stream. Has no effect if objectMode is true.
    writableHighWaterMark number <optional>
    		 Sets highWaterMark for the writable side of the stream. Has no effect if highWaterMark is provided.
    writableObjectMode boolean <optional>
    		 false Sets objectMode for writable side of the stream. Has no effect if objectMode is true.
